Detailed Description

FilterColumn using floating-point weights.

Common parameters for nppiFilterColumn32f functions include:

Parameters
pSrcSource-Image Pointer.
nSrcStepSource-Image Line Step.
pDstDestination-Image Pointer.
nDstStepDestination-Image Line Step.
oROIRegion-of-Interest (ROI).
pKernelPointer to the start address of the kernel coefficient array. Coefficients are expected to be stored in reverse order.
nMaskSizeLength of the linear kernel array.
nAnchorY offset of the kernel origin frame of reference relative to the source pixel.
nppStreamCtxApplication Managed Stream Context.
Returns
Image Data Related Error Codes, ROI Related Error Codes
