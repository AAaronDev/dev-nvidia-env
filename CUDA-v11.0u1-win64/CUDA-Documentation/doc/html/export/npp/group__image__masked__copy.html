Detailed Description

The masked copy primitives have an additional "mask image" input.

The mask controls which pixels within the ROI are copied. For details see Masked Operation.

Common parameters for nppiCopy_CXM functions include:

Parameters
pSrcSource-Image Pointer.
nSrcStepSource-Image Line Step.
pDstDestination-Image Pointer.
nDstStepDestination-Image Line Step.
pMaskMask-Image Pointer.
nMaskStepMask-Image Line Step.
oSizeROIRegion-of-Interest (ROI).
nppStreamCtxApplication Managed Stream Context.
Returns
Image Data Related Error Codes, ROI Related Error Codes
